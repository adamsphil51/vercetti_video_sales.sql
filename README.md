# Vercetti Video Sales Analytics

Relational database design for a multi-branch consumer electronics company.

## Overview

This project models transactional sales, inventory tracking, staff performance, and revenue analytics for a retail video equipment company operating across multiple branches.

The database supports operational reporting and business intelligence use cases.

## Core Entities

- Branches
- Staff
- Clients
- Products
- Sales Transactions
- Inventory Movement Logs

## Business Analytics Covered

- Revenue by branch
- Monthly revenue trends
- Staff performance ranking
- Product-level profitability
- Low inventory monitoring

## Tech Stack

- PostgreSQL
- SQL (DDL, DML, Aggregations, Joins, Indexing)

## Use Case

Designed to simulate real-world retail analytics for BI and Data Analyst roles.
